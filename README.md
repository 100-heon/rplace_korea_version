Reddit 이라는 커뮤니티 사이트에서 제작한 r/place라는 프로젝트를 구현해보았습니다.

구현을 위해
React와 mongoDB를 사용하였습니다.

배포를 위해 koyeb과 MongoDB atlas를 사용하였습니다.

Reddit에서는 5분의 색칠시간 제한이 있지만 저는 지인에게만 공유하여 제한을 두지 않았습니다.

실제 r/place와 같이 그림을 망치는 것이 가능합니다. 덧칠 가능.

https://rplace-ssu-adsl.koyeb.app/

도메인 구입은 부담이 되어 koyeb에서 제공하는 URL을 사용하였습니다.
